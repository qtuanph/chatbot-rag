# 2.1 — Ingestion Workflow

Workflow này biến file upload thành dữ liệu có thể retrieval được.

## Tóm tắt

```text
Upload -> enqueue task -> parse -> store sections -> embed -> index Qdrant -> finalize
```

## Luồng chính

1. `POST /api/v1/upload`
2. backend lưu file vào RustFS
3. tạo row trong `documents`
4. enqueue task ingest
5. worker đọc file và parse nội dung
6. lưu `document_sections`
7. embed và ghi vector vào Qdrant
8. cập nhật document sang `ready`

## Thành phần chính

| Thành phần | File |
|---|---|
| Route upload | `app/modules/documents/router.py` |
| Task orchestration | `app/modules/documents/tasks.py` |
| Ingestion service | `app/modules/documents/ingestion/ingestion_service.py` |
| Embed/index pipeline | `app/modules/documents/ingestion/pipeline.py` |

## Tenant rule

Mọi bước ingest phải giữ `tenant_id` xuyên suốt:

- document row
- section row
- vector payload
- duplicate detection
- retry / rechunk

## Qdrant rule

Qdrant payload chuẩn:

- `tenant_id`
- `document_id`
- `section_id`

Pipeline tự ensure payload indexes cho các field này trước khi ghi.

## Vì sao ingest có thể chậm

Hiện trạng mặc định:

- ingest chạy nền qua Celery worker
- batch ingest dùng config để cân bằng ổn định và throughput
- embedding local mặc định đang đi qua Docker Model Runner

Vì vậy upload xong mà embed/index chưa hoàn tất ngay là hành vi bình thường.

## Progress realtime

Progress tài liệu hiện dùng **SSE với auto-reconnect**:

- backend cập nhật `status`, `status_stage`, `progress_percent`
- backend mở `GET /api/v1/documents/stream`
- frontend subscribe SSE để cập nhật danh sách tài liệu gần realtime

Không dùng polling làm flow chính nữa.

## Rechunk

Rechunk sẽ:

1. đọc lại artifact đã có
2. xóa sections và vectors cũ
3. parse/chunk lại
4. embed/index lại

Không cần upload lại file từ đầu.

## Invariants

- upload không block user chờ parse xong
- `tenant_id` phải đi xuyên suốt
- không chunk tản mát ngoài pipeline chính
- rechunk phải xóa dữ liệu cũ trước khi ghi lại
- document status phải phản ánh đúng tiến độ thực tế
